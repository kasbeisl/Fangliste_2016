Beschreibung:
Fangliste 2015 ist eine Software zum dokumentieren Ihrer Fänge, Erstellt verschiedene Fanglisten,
Verwaltet Fotos, Fang Statistik, Lake Trophy, Server unterstüzung ,
Fisch Kalkulator usw.
Dieses Programm ist ideal für alle Fischer die Ihre Fänge dokumentieren möchten..
